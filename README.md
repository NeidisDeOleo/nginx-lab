🐳 Servidor Web Nginx con Docker Compose
📌 Descripción

Proyecto básico donde se implementa un servidor web con Nginx usando Docker Compose para mostrar una página HTML.

🚀 Ejecución
git clone https://github.com/TU-USUARIO/nginx-lab.git
cd nginx-lab
docker compose up -d

Abrir en el navegador:
http://localhost:8080

⚙️ Tecnologías
Docker
Docker Compose
Nginx
HTML
